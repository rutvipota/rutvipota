### 🤖 Line-Following Robot — EE 201 · [view repo →](https://github.com/rutvipota/line-following-robot)
*C++ · Arduino · KiCad · PID Control · Fusion 360*

Built an autonomous line-following robot using an Arduino Mega, Adafruit Motor Shield, and a 7-photoresistor sensor array. Implemented a closed-loop PID controller (P=6, I=0, D=16) that computed a weighted positional error from sensor readings and adjusted dual DC motor speeds in real time to keep the robot centered on the track.

Designed a custom 3D-printed chassis in Fusion 360 with dedicated mounting for all components and a low-centered battery holder for stability. Transferred the breadboard sensor circuit onto a custom KiCad PCB with soldered photoresistors, LEDs, and voltage dividers for improved reliability. Added a cardboard light shield and 7 LED illuminators to stabilize sensor readings against ambient lighting variation, and a separate 9V battery to prevent Arduino brownout at higher motor speeds.
